Documentation API

Example requests and responses

The URI of the resource involved is accordingly: http://localhost:3000/

Resource Request verb Description Status code Posted data Response format
/phones.db GET Retrieve all products 200 OK - application/json
/phones.db/1 GET Get a product with ID 1 200 OK - application/json
/phones.db POST Create a new product 201 Created brand, model, os, image, screensize application/json
/phones.db/1 PUT Update product with ID 1 204 No Content brand, model, os, image, screensize application/json
/phones.db/1 DELETE Delete product with ID 1 204 No Content - application/json

GET/phones.db/id

Retrieve a specific product from the database with productID: 'id'.
Returns an HTTP response code of '200 OK' if the resource was succesfully found on the server, along with the response body.
In case the resource is not found on the server the API returns an HTTP response code of '404 (Not Found)'.
If the GET request is not correctly formed then the server will return an HTTP response code of '400 (BAD REQUEST)'.

                    
{
    "data": {
        "id": 1,
        "brand": "Fairphone"
        "model": "FP3",
        "os": "Android",
        "image":"https://upload.wikimedia.org/wikipedia/commons/thumb/c/c4/Fairphone_3_modules_on_display.jpg/320px-Fairphone_3_modules_on_display.jpg",
        "screensize": 5.65
    }
}

POST/phones.db

Add a product to the database.
Returns an HTTP response code of '201 Created' if the product was succesfully added to the database and a response body with the location of the new product.
When one of the categories (brand, model, os, image, screensize) is not defined an HTTP response of '400 Bad Request' will be returned. 

                    
{
  "url": http://localhost:3000/phones.db/2
}

PUT/phones.db/id

Update a product in the database with productID: 'id'.
Returns a status code of '204 No Content' when the product is succesfully updated in the database.

DELETE/phones.db/id

Delete a product from the database with productID: 'id'.
Returns a status code of '204 No Content' when the product is succesfully deleted from the database.
Return a status code of'404 Not Found' when the product with the specified id is not present in the database.
